   └─ Some commands use status_ok(), others use echo() with colors
   └─ Some use raw emoji instead of ICON_* constants

   └─ Confusing for non-Indonesian speakers
   └─ Fix: Standardize all text to English

   └─ Some commands have title in wrong context
   └─ Some don't have title at all
   └─ Fix: Always box_title() at command output start
